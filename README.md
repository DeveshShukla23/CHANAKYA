## End-to-End Data Science Capstone Project

100% original Indian e-commerce dataset | 12 Layers | Rs. 2.67 Crore Revenue Data

## Project Structure

| Layer | Topic | Tech |
|-------|-------|------|
| Layer 1 | Data Source + Simulation | Python, Faker |
| Layer 2 | ETL Pipeline | Pandas, Logging |
| Layer 3 | SQL Data Warehouse | MySQL, Star Schema |
| Layer 4 | EDA | Matplotlib, Seaborn |
| Layer 5 | RFM + Segmentation | Pandas, Animation |
| Layer 6 | Anomaly Detection | Z-Score, Isolation Forest |
| Layer 7 | ML Demand Forecasting | Scikit-learn, Gradient Boosting |
| Layer 8 | Deep Learning | TensorFlow, LSTM |
| Layer 9 | Churn Prediction | Logistic Regression, ROC-AUC |
| Layer 10 | Power BI Dashboard | DAX, Power BI |
| Layer 11 | ARTHA Agentic AI | Groq, LLaMA 3.3 70B |
| Layer 12 | Streamlit Deployment | Streamlit, Plotly |

## Key Highlights
- 1,000 customers | 5,010 orders | Rs. 2.67 Crore revenue
- 34 fraud accounts detected
- 477 churn risk customers identified
- ARTHA - Agentic AI powered by LLaMA 3.3 70B
